# Front-End Studies

Repositório contendo projetos e exercícios desenvolvidos em **HTML, CSS e JavaScript**, focando em desenvolvimento de interfaces web e interatividade no navegador.

## 📚 Conteúdo
- Estruturação de páginas com HTML
- Estilização com CSS
- Manipulação do DOM com JavaScript
- Pequenos projetos front-end

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript

## 🎯 Objetivo
Aprender conceitos de desenvolvimento front-end e construção de interfaces web.

## ▶ Como executar

Abra o arquivo `arquivo.html` em qualquer navegador.
